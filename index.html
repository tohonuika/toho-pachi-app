台シミュレーション診断アプリ

① 台スペック入力

このシートの台に共通する「大当り分母・継続率・1回転あたりの投資（玉）」を入力してください。
モンテカルロ試行回数はサーバ側で 8,000 回 に固定しています。
例: 4円パチで 1000円あたり14回転なら、1回転あたり ≒ 18玉。

② 日ごとのデータ貼り付け

各テキストボックスに、その日の台データを貼り付けてください（タブ or スペース区切り）。
1行 = 1台ぶん。ヘッダー行（台番号〜）はあっても構いません。

1行のフォーマット:
台番号 / 大当り / 大当り確率 / スタート / 総回転数 / 前日最終スタート / 初当り / 最大持玉 / 差玉 / 1セット最大出玉 / 最大連チャン数
7日前 一番古い日

当日(途中まで) 昼から打つとき用
※ 当日の枠 (当日(途中まで)) に1行でもデータがある場合、Monte Carlo の対象はその当日の台だけになります。
※ 当日の枠が空の場合は、7日前〜1日前の台だけで Monte Carlo を行います。
※ どちらの場合も、平均出玉などの推定には過去7日＋当日のデータをすべて使用します。
